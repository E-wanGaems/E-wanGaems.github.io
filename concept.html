Current Concept


The unnatural

The main idea for this game is to play as a character of some type that is born with abnormal abilities. These abilities can be used in different ways and you must decide which way your character uses them. This is done by giving the player choices and changing the story based on the decisions. My first thought for this character would be a duck born with the ability to change it's size. I drew a quick sketch but still debating the idea.
