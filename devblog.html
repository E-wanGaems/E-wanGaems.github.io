Chasing Data

The remake: 24/10/2016

Recently development has been slowed to a crawling point on chasing data. This is because we have decided to completely remake the game. This decision has been made mainly due to the belief that the game can be so much more. When we are talking about a remake we aren't just talking about a graphics remake, we are talking about an entire rewrite of the game mechanics and how it works hence why it is taking a while. One of the things we have been working on is diversity as currently there is only one type of game mode available and based on feedback it isn't the most interesting, so a re-work of the mechanics would allow us more control and make it easier to create new game modes and puzzles.

Another thing we have been working is graphical improvements. The original game lacked animations and a lot of the time looked like still images moving. To make the game look and feel a bit more alive we will be adding a lot more animations to our sprites. These are only a fraction of the things we have been working on but we don't want to reveal too much at this point as not a whole lot is done, we should be releasing the new version in intervals due to a lot of the features having a large amount of time spent on them.
